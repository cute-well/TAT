## Packages
framer-motion | Page transitions and scroll-triggered animations for a premium feel
date-fns | Human-readable date formatting for document timestamps
lucide-react | Beautiful, consistent iconography across the dashboard
clsx | Conditional class name merging
tailwind-merge | Merging tailwind classes safely

## Notes
- Categories and Documents are fetched via REST API
- Process Document simulates an OCR/AI extraction workflow by sending raw text to the backend
- Extracted data is rendered dynamically based on the JSON structure from the backend
